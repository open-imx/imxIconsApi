Welcome to the IMX Icon Library

IMX is a specialized data format created to describe the Dutch railway system with a strong emphasis on geographical details, rather than relying on traditional schematic diagrams to represent the network.

Traditional schematic diagrams use various icons to represent components like signals, switches, and tracks. To make IMX data more user-friendly, this service provides a way to generate an icon for any IMX object based on its properties. This means the same set of icons can be used consistently across Geographic Information Systems (GIS) and web applications, resulting in a more intuitive and unified visual representation of railway systems.
